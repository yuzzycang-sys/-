## 提示词 1：整体页面框架（主页面）

```
设计一个后台数据报表页面，中文界面，1440x900 尺寸，浅色主题。

整体布局分为三个区域：
1. 顶部导航栏（高度 48px，白色背景，底部 1px 灰色分割线）
   - 左侧：蓝色圆形 logo + "鲸鱼座" 品牌名（蓝色 #1890ff 粗体）
   - 中间：水平导航菜单项（首页、数据报表、素材管理、资产管理、工具、智能实验室、直播运营、管理中心、新项目），其中"数据报表"为当前激活态（蓝色文字+底部蓝色下划线）
   - 右侧：环境标签"正式环境" + 用户名"董惠娟"

2. 左侧边栏（宽度 120px，白色背景，右侧 1px 灰色分割线）
   - 可折叠的平台列表：快手（展开）、腾讯、头条
   - 快手展开后显示二级菜单"广告分析"，下有子项：腾讯、头条（蓝色激活态）、快手

3. 右侧主内容区（白色背景，flex 填充剩余空间）
   - 从上到下依次包含：视图栏、公共筛选栏、快捷标签栏、子表标签栏、子表工具栏、数据表格、分页器
   - 具体内容见后续提示词

主色调：#1890ff（蓝色），文字颜色 #333，次要文字 #666，边框 #e8e8e8
字体：PingFang SC / Microsoft YaHei，正文 13px，小字 12px
```

## 提示词 2：视图栏（View Bar）

```
在主内容区顶部设计一个视图操作栏，水平排列，高度约 40px，底部有灰色分割线。

从左到右包含以下元素：

1.【选择视图】下拉按钮
   - 样式：带搜索图标 + "选择视图" 占位文字 + 下拉箭头，边框圆角 4px
   - 默认状态：灰色文字"选择视图"（表示未选中任何视图）
   - 选中状态：显示视图名称，如"大盘·周投"

2.【保存】蓝色文字链接

3.【置顶图标】蓝色实心图钉 📌

4.【置顶视图标签】水平排列的视图标签按钮：
   - "乐乐·周报"、"大盘·周投"、"大盘·月报"
   - 默认态：灰色边框 + 灰色文字 + #fafafa 背景
   - 选中态：蓝色边框 + 蓝色文字 + #e6f7ff 浅蓝背景
   - 点击可选中/取消选中
```

## 提示词 3：视图选择器下拉面板（弹出状态）

```
设计一个从"选择视图"按钮下方弹出的下拉面板，宽度 350px，白色背景，圆角 8px，带阴影。

结构从上到下：

1.【Tab 栏】4 个等宽 Tab：所有 | 我的 | 置顶 | 共享
   - 当前激活 Tab：蓝色文字 + 底部 2px 蓝色下划线
   - 非激活：灰色文字

2.【搜索框】灰色背景输入框，带搜索图标，placeholder"搜索视图名称"

3.【视图列表】可滚动区域，最大高度 260px
   每行视图项包含：
   - 左侧：视图名称（如"乐乐·周报"）
   - 中间：归属标签（"共享"tab 下显示共享人如"陈路遥"，公共视图显示"公共"，灰色小字）
   - 右侧：图钉 icon（点击可置顶/取消置顶）
   - 当前选中行：浅蓝背景 #e6f7ff + 蓝色勾选图标 + 蓝色文字
   - hover 行：#f8f8f8 背景

   示例视图数据：
   - 乐乐·周报（我的，已置顶）
   - 大盘·周投（我的，已置顶）
   - 大盘·月报（我的，已置顶）
   - 归因GAP（我的）
   - 腾讯主包GAP（我的）
   - 捕鱼大咖iOS当天数据（陈路遥，共享）
   - 异常监控视图（陈路遥，共享）
   - 市场大盘（公共）
   - 渠道大盘（公共）

4.【底部状态栏】灰色小字"已置顶 3/10"，右对齐

额外状态：置顶已满 10 个时，未置顶视图的图钉 icon 变灰色不可点击，hover 时显示 tooltip "置顶数量已达上限 10"
空状态：该 Tab 下无视图时显示"暂无视图"居中灰色文字
```

## 提示词 5：更新当前视图弹窗（Modal）

```
设计一个居中弹窗（Modal），宽度 440px，白色背景，圆角 8px，带半透明黑色遮罩。

弹窗结构：

1.【标题栏】"更新当前视图" + 右侧关闭按钮 ✕，底部分割线

2.【视图名称】
   - 标签"视图名称"（12px 加粗）
   - 输入框（回显当前视图名称如"大盘·周投"，可编辑）
   - 右下角字数计数"8/30"（灰色小字）
   - 超过 30 字符时：输入框边框变红，显示红色错误提示"视图名称上限 30 个字符（中文算 2 个）"

3.【时间选择】
   - 标签"时间选择"
   - Tab 切换栏：相对时间 | 绝对时间
   - 相对时间（默认）：4 个可选标签按钮"今天""昨天""近7天""近30天"，选中态蓝色边框+浅蓝背景
   - 绝对时间：两个日期输入框，中间用"~"连接

4.【底部按钮】右对齐
   - "取消"灰色边框按钮
   - "确定"蓝色实心按钮（名称为空或超限时置灰不可点击）

请设计两个状态变体：
- 变体 A：正常填写状态
- 变体 B：名称超限报错状态
```

## 原型交互连接说明

在 Figma 中设置以下原型交互：

1. 点击"选择视图" → 弹出视图选择器面板（overlay）
2. 点击视图选择器中的视图行 → 关闭面板，视图栏更新选中状态
3. 点击"保存" → 弹出保存菜单（overlay）
4. 点击"更新当前视图" → 关闭菜单，弹出更新弹窗（overlay）
5. 点击"另存为新视图" → 关闭菜单，弹出另存弹窗（overlay）
6. 点击弹窗"确定" → 关闭弹窗，回到主页面
7. 点击弹窗"取消"或 ✕ → 关闭弹窗
8. 点击"所有筛选" → 弹出筛选项浮窗（overlay）
9. 点击时间范围区域 → 弹出日期选择器（overlay）
10. 点击日期选择器"确定" → 关闭选择器
11. 点击 sheet 标签 → 切换标签激活状态
12. 点击置顶视图标签 → 切换选中/取消选中状态
13. 点击时度按钮（日/周/月） → 切换激活状态
